# activity-2
Activity 2: pseudocode to python program for median calculations.
## Objective
convert the given pseudo code that sorrts an array of numbers and find the median.
-----
Description:
This program implements:
-A sorting algorithm (bubble sort)
- A function to calculate the median of numbers


  #features
  - Sorts numbers in Ascending order
  - Find median For:
  - Odd number of elements
  - Even number of elements

# How it works
1. The numbers are sorted using bubble sort
2. If the  number is even , then the avg of two middle numbers is returned.
3. If the number is odd , the middle number is returned.
